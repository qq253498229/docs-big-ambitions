定价

  • 暂停游戏

  • 选择门店

  • 临时关闭

  • 切换到【库存与定价】标签页（如果之前就在这个标签页，要切换到别的再切回来，否则价格会缓存不变）

  • 观察市场最低价

  • 将定价设置为最低价少1分（如果最低价为12.34，那么我们就设置成12.33）

  • 最后别忘了把门店开张！！！

注意，这个是正常玩法的定价，核弹店不在讨论范围内
